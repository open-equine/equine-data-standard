## What Is This?

The **Equine Profile Standard** defines how a horse's identity, pedigree, ownership, media, metadata, and consent data should be structured for digital identity, AI indexing, and global interoperability. This is minimal viable basic horse profile. 

---

## Who Is This For?

Developers, Equine Platforms and Institutions building modern technology for the equine sector.

## Domains

| Domain | Description |
|--------|-------------|
| Horse Identity | Core identification fields |
| Pedigree | Lineage and bloodline |
| Owner & Breeder | Ownership and stable information |
| Media | Visual media assets linked to a horse profile. Detailed media specification is addressed separately. |
| Meta | Data capture metadata |
| Consent | Data sharing acknowledgement |
